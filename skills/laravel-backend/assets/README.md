# Assets (Шаблоны)

Готовые шаблоны для быстрого создания компонентов Laravel приложения.

## Доступные шаблоны

### migration-template.php

Шаблон для создания миграций базы данных.

**Содержит:**
- Примеры всех основных типов полей
- Примеры внешних ключей
- Примеры индексов
- Комментарии с объяснениями

**Использование:**
1. Создайте миграцию: `./artisan make:migration create_[table_name]_table`
2. Скопируйте нужные части из шаблона
3. Замените `[table_name]` на имя вашей таблицы

### model-template.php

Шаблон для создания Eloquent моделей.

**Содержит:**
- PHPDoc блок с @property аннотациями
- Примеры fillable и hidden полей
- Примеры casts
- Примеры relationships (hasMany, belongsTo, etc.)
- Примеры scopes
- Примеры accessors и mutators

**Использование:**
1. Создайте файл модели в `php/app/Shared/Models/`
2. Скопируйте шаблон
3. Замените `[ModelName]` на имя вашей модели
4. Добавьте необходимые поля и отношения

### service-template.php

Шаблон для создания сервисов.

**Содержит:**
- Базовую структуру сервиса
- Примеры dependency injection
- Комментарии по использованию
- Информацию о Service Providers

**Использование:**
1. Создайте директорию в `php/app/Shared/Services/[ServiceName]/`
2. Создайте файл `[ServiceName].php`
3. Скопируйте шаблон
4. Замените `[ServiceName]` на имя вашего сервиса
5. Реализуйте необходимую логику

## Советы

- **Всегда адаптируйте шаблоны** под конкретные нужды проекта
- **Удаляйте неиспользуемый код** из шаблонов
- **Следуйте стандартам проекта** при именовании и структуре
- **Добавляйте PHPDoc** для всех публичных методов и свойств
